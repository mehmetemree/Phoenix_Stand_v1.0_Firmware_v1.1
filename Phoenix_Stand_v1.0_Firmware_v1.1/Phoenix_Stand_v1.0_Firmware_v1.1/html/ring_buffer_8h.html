◆ RING_BUFFER_MASK

#define RING_BUFFER_MASK   (RING_BUFFER_SIZE-1)

Used as a modulo operator as a % b = (a & (b - 1)) where a is a positive index in the buffer and b is the (power of two) size of the buffer.

◆ RING_BUFFER_SIZE

#define RING_BUFFER_SIZE   32

The size of a ring buffer. Due to the design only RING_BUFFER_SIZE-1 items can be contained in the buffer. The buffer size must be a power of two.
